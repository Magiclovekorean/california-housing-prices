# Predict prices of house in california using linear regression

## Setup 
Run the autoSetup.sh script that does all the instructions by running:
```bash
chmod +x autoSetup.sh
./autoSetup.sh
```
### Alternatively, if you want to do it manually:


Create a python virtual enviroment with 

```bash
python -m venv venv
source venv/bin/activate
```

Then, install the required packages running

```bash
pip install -r requirements.txt
```
